常见问题




1. 软件路径错误




问题：运行命令时提示找不到软件
解决：检查 Inputfile 中的软件路径设置，确保使用绝对路径








2. 训练失败




问题：训练过程中出错
解决：
- 检查训练集文件格式是否正确
- 确认初始势函数文件存在
- 检查元素映射是否正确








3. VASP 计算设置错误




问题：VASP 计算无法运行
解决：
- 检查 input.yaml 中的 VASP 参数
- 确认 POTCAR 文件路径正确
- 检查作业提交脚本配置








4. 数据集生成问题




问题：Extxyz 数据集生成失败
解决：
- 检查 OUTCAR 文件是否完整
- 确认输入文件路径正确
- 检查 YAML 配置文件格式
