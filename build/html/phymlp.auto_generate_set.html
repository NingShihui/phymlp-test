phymlp.auto_generate_set.config module

配置管理模块

该模块负责加载和管理整个项目的配置信息，包括Materials Project API配置、 路径设置、形变参数、微扰参数、VASP计算参数等。

phymlp.auto_generate_set.config.load_config(config_file='input.yaml')[源代码]

加载和验证配置文件。

参数:

config_file (str) -- 配置文件的路径，默认为 'input.yaml'。

返回:

包含所有配置参数的字典。

返回类型:

dict

抛出:

  • FileNotFoundError -- 当指定的配置文件不存在时抛出。

  • yaml.YAMLError -- 当配置文件格式错误时抛出。

示例

>>> config = load_config("input.yaml")
>>> api_key = config['api']['key']
>>> elements = config['elements']
