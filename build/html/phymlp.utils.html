class phymlp.utils.core.RUN[源代码]

基类：PUBLIC

运行管理类

负责训练和计算任务的执行管理，包括SLURM作业提交和监控。

示例

run = RUN() # 读取配置 input_info = run.Read("Inputfile") run.Key_and_parament(input_info) # 执行训练 train_drive = run.Bool_commands(run.train) train_dir, train_cmd = run.Train_commands(train_drive) run.Train(train_drive, train_dir, train_cmd)
