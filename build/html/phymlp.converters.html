phymlp.converters.cfg2extxyz module

CFG to EXTXYZ格式转换器

本模块提供将MTP结构CFG文件转换为EXTXYZ格式的功能。 支持从CFG文件中提取原子结构、能量、应力等信息。

phymlp.converters.cfg2extxyz.parse_cfg_file(file_path)[源代码]

解析包含多个CFG结构的文件

参数:

file_path (str) -- CFG文件路径

返回:

包含所有配置信息的字典列表，每个字典代表一个结构

返回类型:

list

抛出:

  • FileNotFoundError -- 当指定的文件不存在时

  • ValueError -- 当文件格式不正确时

示例

>>> configs = parse_cfg_file("train.cfg")
>>> print(f"找到 {len(configs)} 个结构")
找到 100 个结构
phymlp.converters.cfg2extxyz.write_extxyz(configs, output_file, type_map=None, float_fmt='.9f')[源代码]

将解析后的配置写入EXTXYZ文件

参数:

  • configs (list) -- parse_cfg_file()返回的配置列表

  • output_file (str) -- 输出文件路径

  • type_map (dict, optional) -- 原子类型到元素符号的映射字典

  • float_fmt (str, optional) -- 浮点数格式字符串，默认为".9f"

抛出:

  • IOError -- 当无法写入输出文件时

  • KeyError -- 当type_map中缺少必要的原子类型映射时

示例

>>> configs = parse_cfg_file("train.cfg")
>>> type_map = {0: 'Ni', 1: 'Cr', 2: 'Fe'}
>>> write_extxyz(configs, "output.xyz", type_map=type_map)
cfg格式到extxyz格式转换完成！

phymlp.converters.outcar_converter module

OUTCAR到EXTXYZ格式转换器

本模块提供从VASP OUTCAR文件中提取结构并转换为EXTXYZ格式的功能。 支持按步长提取结构，保留原子位置、能量、晶格等信息。

phymlp.converters.outcar_converter.extract_structures_from_outcar(outcar_file, output_file, step=1)[源代码]

从OUTCAR文件中提取结构并保存为extxyz文件

参数:

  • outcar_file (str) -- OUTCAR文件路径

  • output_file (str) -- 输出的extxyz文件路径

  • step (int, optional) -- 提取结构的步长，默认为1

抛出:

  • FileNotFoundError -- 当OUTCAR文件不存在时

  • ValueError -- 当OUTCAR文件格式不正确时

示例

>>> # 提取所有结构
>>> extract_structures_from_outcar("OUTCAR", "all_structures.xyz")
>>> # 每10步提取一个结构
>>> extract_structures_from_outcar("OUTCAR", "sparse_structures.xyz", step=10)
